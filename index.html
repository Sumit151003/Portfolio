Mini-Amazon

The Mini-Amazon project is a basic HTML and CSS clone of the Amazon website, focusing on replicating its layout and core features such as navigation, search, and product display. Through responsive design techniques like flexbox and grid layout, it ensures compatibility across devices. While lacking dynamic functionalities, it serves as an excellent learning tool for beginners to grasp front-end development concepts and practice building user interfaces. Visit website
